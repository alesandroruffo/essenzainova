# Essenza Inova — Landing Page

Landing page de alta conversão para o **Curso Essenza**, com foco em captação de leads e reservas no WhatsApp.

## Como executar localmente

```bash
python3 -m http.server 4173
```

Depois, abra: `http://localhost:4173`

## Estrutura

- `index.html`: estrutura da landing page e conteúdo.
- `styles.css`: identidade visual, layout responsivo e componentes.
- `script.js`: interações (menu mobile, modal dos reels e eventos de analytics).

## Eventos de analytics implementados

- `cta_whatsapp_click_header`
- `cta_whatsapp_click_hero`
- `cta_whatsapp_click_final`
- `open_turma_modal`
- `click_assistir_reels_turma_X`
